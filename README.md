# Heart Disease Prediction App ❤️

An interactive web application built with **Streamlit** that predicts the likelihood of heart disease using Machine Learning.

## 🚀 Features
- **Single Prediction:** Input patient data manually.
- **Bulk Prediction:** Upload a CSV file for multiple results at once.
- **Multiple Models:** Choose between Logistic Regression, SVM, Decision Tree, and Random Forest.

## 🛠️ Tech Stack
- **Language:** Python
- **Frontend:** Streamlit
- **ML Libraries:** Scikit-learn, Pandas, Numpy
- **Deployment:** Pickle for model serialization
